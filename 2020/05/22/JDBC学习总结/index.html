JDBC学习总结

发表于 2020-05-22 | 分类于 JavaSe

JDBC:

JDBC概念:

 JDBC: Java DataBase Connectivity (Java数据库连接)JDBC是Java访问数据库的标准规范

  作用:用于执行SQL语句的Java API(Java语言通过JDBC可以操作数据库);

 好处:
  1.我们只需要会调用JDBC接口中的方法即可,使用简单
  2.使用同一套java代码,进行少量的修改就可以访问其他JDBC支持的数据库

JDBC使用步骤

  JDBC的编程流程步骤:
1.导入相关jar包 直接在lib包下右击找到Add as Library 加载到当前环境

2.注册驱动(利用反射Class.forName(“”))

3.获取Connection连接对象

4.编写sql语句

5.执行sql语句

6.处理结果集

7.释放资源
