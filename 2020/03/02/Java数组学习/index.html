Java数组学习

发表于 2020-03-02 | 分类于 JavaSe

数组

数组的基本概念：

数组： 就是同一种数据类型的集合（也可以说是容器）；
为什么要用数组（好处）：读取数据比较方便；

数组的特点：

  1. 数组是一种引用数据类型；
  2. 数组当中的多个数据，类型必须统一；
  3. 数组的长度在程序运行期间是不可以改变的；

    数组的定义格式（直接推荐一）：

    一. 数据类型 [] 数组名 例如：int [] arr;
    二. 数据类型 数组名 [] 例如： int arr [];

    数组静态初始化：

    不指定数组的直接长度，长度由我们定义的时候同时给予的元素值的个数来决定

格式：

数据类型 [] arr = {1,3,5,6,7};

数组的常见异常：

java.lang.NullPointerException: 空指向异常
如 赋予数组为null；

java.lang.ArrayIndexOutOfBoundsException:数组越界异常
如： 读取比该数组的length相等或着大于的时候（数组的小标从0开始，所以如果数组的长度为6，那么最大的下表为6 即 0，1，2，3，4，5）；
