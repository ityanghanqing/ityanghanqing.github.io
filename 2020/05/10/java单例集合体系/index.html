java单例集合体系

发表于 2020-05-10 | 分类于 JavaSe

Collection

 集合: 集合是java中提供的一种容器.可以用来存储多个数据
 集合与数组都是容器,它们有什么区别:
1.集合的长度是可变的;数组的长度是固定的
2.集合存储的都是引用类型的数据,如果想存基本数据类型,存储的是其相对应的包装类; 数组中存的是同一个类型的元素,可以存储任意数据类型;
 Collection 是单例集合的跟接口 常用子接口有List与Set
Collection是所有单例集合的跟接口,所以在Collection中定义了单例集合(List与Set)通用的一些方法,这些方法用于操作所有的单列集合 add(E e) clear() remove(E e) isEmpty() size() toArray()等方法;

List接口

 List是Collection下的子接口,常用实现类为ArrayList与LinkedList
 ArrList底层是一个动态数组存储,有索引,允许null值,默认容量是10,特点就是查询快,增删慢
 LinkedList:双链表,对于set/get操作而言,增删快,查询慢,因为要移动指针;
