Java中的常量与变量

一.标识符

标识符的命名规则
  • 标识符可以由字母、数字、下划线（_）和美元符（$）组成，不能以数字开头
  • 标识符严格区分大小写
  • 标识符不能是Java关键字和保留字
  • 标识符的命名最好能”见明知意”

二.常量:

常量的定义:

 在程序运行过程一直不会改变的量称为常量,通常也称为”final变量”,(也可以理解为是一种特殊的变量),它的值被设定后,在程序运行过程中不允许改变.

  1. 常量名通常使用大写字母,但不是必须的;
  2. 当定义的final变量为成员变量时,必须在定义时就设定它的初始值,否则将会提示编译错误;

三.变量:

变量的定义:

 定义变量就是告诉编译器这个变量的数据类型,这样编译器才知道需要分配多少空间给它,以及它能存什么样的数据类型.在程序运行过程中空间的值是变化的,这个内存空间就称为变量.也称为变量名
 变量名的命名规则:

  1. 变量名必须是一个有效的标识符;
  2. 变量名不可以使用Java关键字;
  3. 变量名不能重复;
  4. 最好还是见明知意;

四.变量与常量的命名规范：

  1. 所有变量，方法，类名：见明知意；
  2. 类成员变量： 首字母小写和驼峰原则:(例如：stuName);
  3. 局部变量： 首字母小写和驼峰原则；
  4. 常量： 大写字母和下划线：（例如： double PI = 3.14）;
  5. 类名：首字母大写和驼峰原则：（例：Student,GoodStudent）;
  6. 方法名： 首字母小写和驼峰原则：（例：jump(),jumpHigh）
