时间类

Date

Date 是一个类,是可以使用关键字new 出来一个对象的,但是其中大部分方法已经启用 Date类中重写了toString方法,输出的是字符串形式的 星期 月 几号 时间 时区缩写 (我国是东八区) 年

Date date = new Date();

DateFormat

DateFormat是一个抽象类, 若要使用就需要使用它的实现类 SimpDateFormat,可以通过这个类帮我们完成日期和文本之间的转换,也就是可以在Date对象与String对象之间进行来回转换

格式化: 按照指定的格式 从Date对象转换为String对象
解析:按照指定的格式,从String对象转换为Date对象

Calendar

Calendar为抽象类,不能直接创建对象,而是通过静态方法创建,返回子类对象 西方星期的开始为周日,中国为周一
在Calendar类中,月份的表示是以0-11代表1-12月 日期是有大小关系的,时间靠后,时间越大; 

Calendar calendar= Calendar.getInstance();
//常用方法
int year = calendar.get(Calendar.YEAR);
//设置年份
calendar.set(Calendar.YEAR,2020);
calendar.add(Calendar.YEAR,-10);
//将日历对象转换为日期对象
Date date = calendar.getTime();
System.out.println(date);//Sun May 09 08:50:50 CST 2010

包装类:

 java中有四类八种基本数据类型,这些基本数据类型都有相对应的包装类
 基本类型包装类的概述: 将基本数据类型封装成对象的好处在于可以在对象中定义更多的功能方法操作该数据,用于基本数据类型与字符串之间的转换;
 在java5(JDK1.5)开始 ,基本类型与包装类的装箱,拆箱动作可以自动完成
 如果字符串参数的内容无法正确转换为对应的基本类型,则会抛出java.lang.NumberFormatException异常

基本数据类型         包装类  
  byte               Bybte
  short              Short
  int                Integer
  long               Long
  float              Float
  double             Double
  boolean            Boolean
  char               Character
